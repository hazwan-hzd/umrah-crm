# 🕋 UmrahCRM - Sistem Pengurusan Jemaah

Production-grade CRM for Umrah travel agencies. Built with vanilla HTML/CSS/JS + Chart.js.

## Features
- Dashboard with KPI cards and 4 Chart.js charts
- 5-column Kanban lead management board
- Pilgrim directory with search, filters, and visa tracking
- 7-stage journey pipeline with progress tracking
- Financial management with payment recording
- Agent performance tracking with client assignments
- WhatsApp message simulator with templates
- Notification center with smart alerts
- Full CRUD operations with localStorage persistence
- Mobile-first responsive design with desktop mode

## Quick Start
```bash
# Clone and serve
git clone https://github.com/hazwan-hzd/umrah-crm.git
cd umrah-crm
python3 -m http.server 8080
# Open http://localhost:8080
```

## Tech Stack
- HTML5 + CSS3 (vanilla, no frameworks)
- JavaScript ES6+ (vanilla)
- Chart.js 4.x (CDN)
- Font Awesome 6.x (CDN)
- Plus Jakarta Sans (Google Fonts)
